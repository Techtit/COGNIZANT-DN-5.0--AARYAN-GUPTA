1. Understanding Recursive Algorithms
What is Recursion?

Recursion is a programming technique where a method calls itself repeatedly until a stopping condition (base case) is reached.

Recursion simplifies problems that can be divided into smaller versions of the same problem.

Advantages of Recursion
Simple and readable code.
Useful for mathematical computations.
Helps solve complex problems efficiently.
Base Case

A base case stops recursive calls and prevents infinite recursion.

5.Optimization

The recursive solution can be optimized using the mathematical formula and Math.pow().

public static double forecastOptimized(double currentValue,
                                       double growthRate,
                                       int years) {

    return currentValue *
            Math.pow(1 + growthRate, years);
}
Optimized Complexity
Method	Time Complexity	Space Complexity
Recursive	O(n)	O(n)
Math.pow()	O(log n)	O(1)
Conclusion

The recursive approach is easy to understand and implement. However, for large forecasting periods, the optimized approach using Math.pow() is preferred because it reduces execution time and memory usage.
